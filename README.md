# Project Name:
  SHREE KALYANI FOODS:
  This project is a web application for a food products business. It allows users to browse, view details, and explore different food items. The app provides a clean and responsive interface to showcase products effectively, making it easier for customers to discover and choose their favorite food items.



## Project Info
- **URL / Demo:** [https://shreekalyanifoods.netlify.app/]

## Technologies Used
This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
