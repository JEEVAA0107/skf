# Shree Kalyani Foods

A modern web application for South Indian food products business. This project allows users to browse products, submit contact forms, and provide feedback with data stored in MongoDB Atlas.

## Features
- 🍛 Product showcase with detailed information
- 📞 Contact form with MongoDB storage
- ⭐ Customer feedback and rating system
- 📱 Responsive design for all devices
- 🔒 Secure data storage with MongoDB Atlas

## Project Info
- **Live Demo:** [https://shreekalyanifoods.netlify.app/]
- **Database:** MongoDB Atlas
- **Backend API:** Node.js + Express

## Technologies Used

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- CORS enabled

## Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/shree-kalyani-foods.git
cd shree-kalyani-foods
```

### 2. Install Dependencies
```bash
# Frontend
npm install

# Backend
cd server
npm install
```

### 3. Environment Setup
Create `.env` file in root directory:
```env
VITE_MONGODB_URI="your_mongodb_connection_string"
VITE_DB_NAME="shree_kalyani_foods"
```

### 4. Run Application
```bash
# Start backend server (Terminal 1)
npm run server

# Start frontend (Terminal 2)
npm run dev
```

## Database Collections
- `contact_messages` - Contact form submissions
